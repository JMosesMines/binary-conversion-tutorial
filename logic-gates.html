Digital Logic Gates

Logic gates are the physical structural building blocks of integrated digital circuits. They execute basic Boolean algebra functions on one or more binary inputs to generate a singular binary output value.

Core Hardware Gate Profiles

Gate Schematic Symbol Boolean Expression Functional Rule
NOT Y = A' Inverts the input value completely.
AND Y = A · B Output is 1 only if ALL inputs are 1.
OR Y = A + B Output is 1 if AT LEAST ONE input is 1.
XOR Y = A ⊕ B Output is 1 if the inputs are DIFFERENT.

Derived Inverted Gates (NAND, NOR)

Placing an inversion bubble at the output node creates NAND and NOR gates, which cleanly flip the default operational results of standard AND and OR gates.
