Understanding Two's Complement

In standard binary, we can only represent positive numbers. To represent negative numbers, digital systems use the Two's Complement framework. This is an elegant design because it allows standard arithmetic hardware to handle addition and subtraction identically without needing separate subtractor circuits.

In a signed 8-bit system, the absolute far-left bit (Most Significant Bit, or MSB) becomes the sign bit. Instead of representing a positive value (+128), it represents a negative value (-128).

The Recipe for Negative Numbers

To convert a negative decimal number to Two's Complement binary format:

  1. Write out the straight binary value of the positive version of the number. Ensure it is padded out to the fixed bit length (e.g., 8 bits).
  2. Invert all bits: Flip every 1 to a 0, and every 0 to a 1. (This is known as the One's Complement).
  3. Add 1: Perform binary addition by adding 1 to your inverted sequence.
